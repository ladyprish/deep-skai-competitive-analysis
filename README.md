# Deep-SKAI Competitive Analysis Tool

Interactive competitive analysis tool comparing Deep-SKAI capabilities against Decision Intelligence Platform competitors.

## Features
- Compare Deep-SKAI against 22 competitors
- View capabilities across Mandatory, Common, and Differentiated categories
- See detailed execution descriptions for each platform
- Consolidated capabilities using consistent terminology

## Deployment Instructions

### Deploy to Vercel
1. Push this repository to GitHub
2. Go to vercel.com
3. Click "New Project"
4. Import your GitHub repository
5. Framework Preset: **Create React App**
6. Click "Deploy"

### Local Development
```bash
npm install
npm start
```

The app will open at http://localhost:3000

## Built With
- React
- Tailwind CSS
- Lucide React Icons

## Updates
To update the tool:
1. Make changes to `src/App.jsx`
2. Commit and push to GitHub
3. Vercel will automatically redeploy
